    Comment trouver les caractéristiques et la configuration système requise d'un ordinateur Windows 10

    Voici comment trouver la configuration requise, les versions et les langues disponibles de l'ordinateur pour Windows 10.

    Configuration requise pour installer Windows 10

    Voici la configuration requise de base pour installer Windows 10 sur un PC. Si votre appareil ne répond pas à cette configuration, vous pourriez ne pas profiter au mieux de Windows 10 et devriez peut-être envisager d'acheter un nouveau PC.
    Processeur :
    Processeur de 1 GHz ou plus rapide ou SoC
    RAM : 
    		 1 gigaoctet (Go) pour système 32 bits ou 2 Go pour système 64 bits 
    Espace sur le disque dur :
    		 16 Go pour système 32 bits ou 32 Go pour système 64 bits
    Carte graphique :
    		 DirectX 9 ou version ultérieure avec pilote WDDM 1.0
    Écran :
    		 800x600
    Connexion Internet :

    Une connexion Internet est nécessaire pour effectuer des mises à jour ainsi que pour télécharger et utiliser certaines fonctionnalités. Windows 10 Professionnel en mode S, Windows 10 Professionnel Éducation en mode S, Windows 10 Éducation en mode S et Windows 10 Entreprise en mode S requièrent une connexion Internet pendant la configuration initiale de l'appareil (en anglais, Out of Box Experience, abrégé OOBE), ainsi qu'un compte Microsoft (MSA) ou Azure Activity Directory (AAD). Une connexion Internet est également requise pour sortir Windows 10 du mode S. En savoir plus le mode S.
    Des exigences supplémentaires peuvent survenir ultérieurement pour les mises à jour, ainsi que pour l'activation de fonctionnalités spécifiques sur le système d'exploitation.

    Maintien de Windows 10 à jour

    Windows 10 est conçu pour fournir les mises à jour pendant toute la durée de vie prise en charge de l'appareil. Deux types de mises à jour peuvent être fournis : les mises à jour de qualité et les mises à jour de fonctionnalité. Les mises à jour de qualité incluent des mises à jour de sécurité et autres et sont généralement publiées une fois par mois. Les mises à jour de fonctionnalités incluent des mises à jour de qualité ainsi que de nouvelles fonctionnalités pour Windows 10 et sont généralement fournies deux fois par an. Il est important de s'assurer que votre appareil reçoit ces mises à jour et est conservé à jour afin d'être protégé. Windows 10 vérifie régulièrement la disponibilité des mises à jour pour vous. Lorsqu'une mise à jour est disponible, s'il y a suffisamment d'espace disque sur votre appareil, elle est automatiquement installée. Afin que Windows 10 reste à jour, il est important de s'assurer que votre appareil dispose de suffisamment d'espace disponible. Reportez-vous aux informations applicables supplémentaires dans les remarques suivantes.
    Remarques importantes sur les mises à jour :
      • Un appareil peut ne pas recevoir les mises à jour si le matériel n'est pas compatible, si des pilotes sont manquants ou s'il n'y a pas suffisamment d'espace disque libre, ou encore si le fabricant d'équipement d'origine n'assure pas le support. Consultez la Fiche technique relative au cycle de vie Windows ou le Forum aux questions sur le cycle de vie pour les produits Windows pour en savoir plus sur la chronologie de fourniture de chaque mise à jour de fonctionnalité.
        • Une partie de l'espace disque nécessaire pour installer les mises à jour est requise uniquement de manière temporaire. En général, dix jours après l'installation d'une mise à jour, un nettoyage de disque est réalisé automatiquement afin de supprimer les copies des fichiers Windows plus anciens et inutiles et libérer de l'espace.
      • Toutes les fonctionnalités d'une mise à jour peuvent ne pas fonctionner sur tous les appareils.
      • Une connexion Internet est requise pour effectuer les mises à jour et des frais d'accès Internet (FAI) peuvent s'appliquer.
      • Si vous avez besoin d'aide pour installer une mise à jour, l'Assistant de mise à jour Windows 10 peut vous aider.

    Plus d'informations sur l'espace disque nécessaire pour les mises à jour

    La taille du système d'exploitation Windows fourni avec votre appareil et la quantité d'espace nécessaire pour télécharger et installer les mises à jour Windows peuvent considérablement varier, car elles dépendent de différents facteurs. Reportez-vous ici pour savoir pourquoi. Les facteurs qui influencent la quantité d'espace disque libre nécessaire pour recevoir une mise à jour incluent : la version de Windows précédemment installée sur la machine, la quantité d'espace disque disponible à réutiliser à partir des fichiers Windows, comme le fichier d'échange de mémoire virtuelle ou le fichier d'hibernation, les applications installées sur votre appareil et la façon dont elles stockent les données. Vous pouvez devoir libérer de l'espace disque, par exemple, en déplaçant des fichiers vers un service de stockage cloud comme OneDrive (application installée sur les appareils Windows 10), en supprimant les fichiers inutiles de votre appareil ou en insérant un lecteur USB pour terminer la mise à jour.
    Si possible, Windows libère automatiquement de l'espace disque s'il n'y en a pas suffisamment de libre (comme le nettoyage disque décrit ci-dessus) et vous guide dans la libération de plus d'espace si le nettoyage automatique n'est pas suffisant. Pour plus d'informations, reportez-vous à Libérer de l'espace pour installer la dernière mise à jour Windows 10 ou consultez le Forum aux questions correspondant.

    Exigences spécifiques aux fonctionnalités de Windows 10

    En plus de la configuration requise ci-dessus qui est nécessaire pour exécuter Windows, d'autres conditions sont requises pour certaines fonctionnalités. Dans certains cas, les fonctionnalités incluses avec les versions mises à jour de Windows 10 seront optimisées avec des processeurs plus récents. Pour obtenir une assistance matérielle spécifique, veuillez vous adresser à votre fabricant d'équipement d'origine (FEO). Vous trouverez ci-après des informations supplémentaires concernant la configuration requise pour les principales fonctionnalités :

    • Le chiffrement de lecteur BitLocker (disponible avec Windows 10 Professionnel ou Windows 10 Entreprise uniquement) requiert un Module de plateforme sécurisée (TPM) 1.2 ou version ultérieure et un BIOS ou une UEFI compatible TCG (Trusted Computing Group). BitLocker peut être utilisé sur des appareils sans TPM, mais vous devrez enregistrer une clé de démarrage sur un appareil supprimable, tel qu'un lecteur USB. La prise en charge de TPM 2.0 et d'InstantGo est requise lorsque vous souhaitez chiffrer automatiquement le disque local lors de la jointure d'un appareil à Azure Active Directory (AAD). Contactez le fabricant de votre PC pour vérifier que votre appareil prend en charge la version TPM correcte et InstantGo pour le scénario que vous souhaitez mettre en place.
    • BitLocker To Go requiert un disque mémoire flash USB (disponible sur Windows 10 Professionnel et Windows 10 Entreprise uniquement).
    • Le client Hyper-V requiert un système 64 bits avec des fonctions de traduction d’adresses de second niveau (SLAT) et 2 Go de RAM supplémentaires (disponbiel sur Windows 10 Professionnel et Windows 10 Entreprise uniquement).
    • Cortana n'est actuellement disponible que sous Windows 10 pour les États-Unis, le Royaume-Uni, la Chine, la France, l'Italie, l'Allemagne, le Brésil, le Mexique, le Japon, le Canada, l'Espagne, l'Australie et l'Inde.
    • Un compte Microsoft est requis pour certaines fonctionnalités.
    • Miracast nécessite une carte vidéo qui prend en charge le modèle WDDM (Windows Display Driver Model) 1.3, ainsi qu'une carte Wi-Fi qui prend en charge Wi-Fi Direct.
    • L'application Films et TV n'est pas disponible dans toutes les régions. Pour accéder à la liste de régions la plus à jour, rendez-vous sur la page d'information Films et TV.
    • Le démarrage sécurisé requiert un microprogramme prenant en charge la norme UEFI v2.3.1 Errata B et disposant de l'autorité de certification Microsoft Windows dans la base de données des signatures UEFI.
    • Skype n'est disponible que dans certains pays et régions. Appels vers certains pays et régions uniquement. Les numéros spéciaux, surtaxés et virtuels ne sont pas inclus. Pour plus d'informations, rendez-vous sur la Page de notre Forum aux questions Office.
    • Ancrer : Le nombre d'applications pouvant être ancrées dépend de la résolution minimale pour l'application avec une limite de deux applications en mode Tablette et quatre applications en mode Bureau.
    • La reconnaissance vocale varie selon le micro de l'appareil. Pour une expérience optimale, vous aurez besoin des éléments suivants :
      • Réseau de microphones haute fidélité
      • Pilote matériel avec géométrie du réseau de microphones exposée
    • Le mode Tablette est disponible sur les tablettes et les appareils 2-en-1 équipés d'indicateurs GPIO ; ceux équipés d'indicateurs de portable/tablette peuvent être configurés de manière à passer automatiquement en « mode tablette ».
    • Fonction tactile : Pour utiliser les fonctionnalités tactiles, vous avez besoin d'une tablette ou d'un moniteur prenant en charge l'interaction tactile multipoint.
    • L'authentification à deux facteurs nécessite l'utilisation d'un code PIN, de la biométrie (lecteur d'empreintes digitales ou caméra infrarouge illuminée) ou d'un téléphone équipé des fonctions Wi-Fi ou Bluetooth.
    • Windows Hello requiert une caméra configurée pour l'imagerie proche infrarouge ou un lecteur d'empreinte digitale pour l'authentification biométrique. Les appareils sans capteurs biométriques peuvent utiliser Windows Hello avec un code confidentiel ou une clé de sécurité compatible Microsoft portable.
    • L'application Xbox nécessite un compte Xbox Live, qui n'est pas disponible dans toutes les régions. Pour accéder à la liste de régions la plus à jour, rendez-vous sur le site Web consacré aux pays et régions Xbox Live.
    • Wi-Fi Direct Printing nécessite une carte Wi-Fi qui prend en charge Wi-Fi Direct, ainsi qu'un appareil qui prend en charge Wi-Fi Direct Printing.

    Versions linguistiques

    Les langues disponibles en localisation complète dans Windows 10 incluent : allemand (Allemagne), anglais (États-Unis), anglais (Royaume-Uni), arabe (Arabie saoudite), bulgare (Bulgarie), chinois (RPC), chinois (Taïwan), coréen (Corée), croate (Croatie), danois (Danemark), espagnol (Espagne), espagnol (Mexique), estonien (Estonie), finnois (Finlande), français (France), français (Canada), grec (Grèce), hébreu (Israël), hongrois (Hongrie), italien (Italie), japonais (Japon), letton (Lettonie), lituanien (Lituanie), néerlandais (Pays-Bas), norvégien, bokmål (Norvège), polonais (Pologne), portugais (Brésil), portugais (Portugal), roumain (Roumanie), russe (Russie), serbe (latin, Serbie), slovaque (Slovaquie), slovène (Slovénie), suédois (Suède), tchèque (République tchèque), thaï (Thaïlande), turc (Turquie), ukrainien (Ukraine).
     
    Autres langues disponibles en tant que Packs linguistiques LIP (Language Interface Pack)

    Abandons et suppressions de fonctionnalités

    Lors de la mise à niveau vers Windows 10 à partir d'une version antérieure de Windows, telle que Windows 7 ou Windows 8.1, ainsi que lors de l'installation d'une mise à jour récente de Windows 10, certaines fonctionnalités peuvent être retirées. Reportez-vous ci-dessous pour plus d'informations sur certaines fonctionnalités clés supprimées :
    • Application de messagerie de bureau : l'application de messagerie sur ordinateur de bureau dispose d'une fonctionnalité de synchronisation qui peut être utilisée pour synchroniser les SMS reçus de Windows Mobile et en garder une copie sur l'ordinateur de bureau. Depuis la mise à jour de mai 2019 (Windows 10, version 1903), la fonctionnalité de synchronisation a été supprimée de tous les appareils. En raison de ce changement, vous pourrez uniquement accéder aux messages de l'appareil qui a reçu le message.
    • Wi-Fi protégé par WEP et TKIP : depuis la mise à jour de mai 2019 (Windows 10, version 1903), un message d'avertissement apparaît lors de la connexion aux réseaux Wi-Fi sécurisés par WEP ou TKIP, qui ne sont pas aussi sûrs que ceux utilisant WPA2 ou WPA3. Dans une prochaine version, toute connexion à un réseau Wi-Fi utilisant ces anciens chiffrements sera interdite. Les routeurs Wi-Fi doivent être mis à jour pour utiliser les chiffrements AES, disponibles avec WPA2 ou WPA3.
    • Cortana : Cortana a été mise à jour et améliorée dans la Mise à jour de mai 2020 de Windows 10 (Windows 10, version 2004). Ces changements impliquent que certaines compétences grand public, telles que la musique, la maison connectée et les compétences de tiers, ne seront plus disponibles. Obtenez plus informations ici.
    • Assistant Mobile : à partir de la mise à jour d'octobre 2018 (Windows 10, version 1809), l'Assistant Mobile est supprimé de votre PC. Utilisez la page Téléphone de l'application Paramètres pour synchroniser votre téléphone portable avec votre PC. Elle comprend toutes les fonctionnalités Assistant Mobile.
    • Groupement résidentiel : la fonctionnalité Groupement résidentiel a été supprimée une lors de la mise à jour d'avril 2018 (Windows 10, version 1803), mais vous pouvez toujours partager des imprimantes, des fichiers et des dossiers. Si vous effectuez une mise à jour à partir d'une version précédente de Windows 10, vous ne verrez plus Groupement résidentiel dans l'Explorateur de fichiers, le Panneau de configuration ou Dépanner (Paramètres > Mise à jour et sécurité > Dépanner). Les imprimantes, fichiers et dossiers que vous avez partagés à l'aide de Groupement résidentiel seront toujours partagés. Au lieu d'utiliser Groupement résidentiel, vous pouvez désormais partager des imprimantes, des fichiers et des dossiers à l'aide des fonctionnalités intégrées à Windows 10 :
      • Partage de vos imprimantes réseau
      • Partage de fichiers dans l’Explorateur de fichiers
      • Pour les utilisateurs de Groupement résidentiel et Xbox 360, veuillez trouver plus d'informations sur la diffusion en continu
    • Application Contacts : dans Windows 10, l'application Contacts affiche les courriers des contacts et des contacts Microsoft 365 de votre établissement pédagogique ou professionnel dans Conversations. Une fois la mise à jour d'avril 2018 (Windows 10, version 1803) effectuée, afin de voir les nouveaux courriers de ces contacts spécifiques dans l'application Contacts, vous devez être connecté et posséder un compte Microsoft 365 ou, pour les comptes d'établissements professionnels ou pédagogiques, par le biais des applications Courrier, Contacts ou Calendrier. Veuillez noter que vous verrez uniquement les courriers pour les comptes d'établissements professionnels ou pédagogiques et certains comptes Microsoft 365.
    • Application Lecteur : l'application Lecteur a été supprimée de Windows 10 lors de la mise à jour Fall Creators Update (Windows 10, version 1709). Microsoft Edge est l'application de remplacement recommandée pour la lecture de fichiers PDF et offre un fonctionnement similaire, ainsi que des fonctionnalités supplémentaires comprenant le support d'accessibilité améliorée, l'entrée manuscrite améliorée et l'assistance pour Demander à Cortana. De la même façon, la Visionneuse XPS Windows est recommandée pour la lecture de fichiers XPS et l'application Photos Windows pour l'affichage de fichiers TIFF. Veuillez noter que les utilisateurs des versions précédentes de Windows 10 peuvent continuer à utiliser l'application Lecteur.
    • Journal Windows : Journal Windows a été supprimé lors de la mise à jour anniversaire Windows 10 (Windows 10, version 1607). Une fois Journal Windows supprimé, vous ne pourrez plus ouvrir ou éditer des fichiers journaux (avec les extensions .JNT ou .JTP). À la place de Journal Windows, nous vous encourageons à passer à OneNote. Si vous devez ouvrir ou éditer vos fichiers journaux, vous trouverez de plus amples informations ici.
    • Gestion des droits numériques Windows Media (WMDRM) : WMDRM n'est plus prise en charge depuis la mise à jour anniversaire Windows 10 (Windows 10, version 1607). Vous ne pouvez plus écouter de la musique ou lire des fichiers vidéo protégés par cette technologie de gestion des droits. Cliquez ici pour en savoir plus.

    Les modifications suivantes ont une incidence sur les appareils qui passent de Windows 7, Windows 8 ou Windows 8.1 à Windows 10 :

    • Windows Media Center : Si vous possédez Windows 7 Édition Familiale Premium, Windows 7 Professionnel, Windows 7 Édition Intégrale, Windows 8 Professionnel avec Media Center ou Windows 8.1 Professionnel avec Media Center et que vous installez Windows 10, Windows Media Center sera supprimé.
    • Gestion des périphériques mobiles (GPM) : la fonctionnalité GPM ne sera pas disponible dans Windows 10 Famille.
    • Gadgets de bureau Windows 7 : Ils sont supprimés au cours de l'installation de Windows 10.
    • Solitaire, Démineur et Dame de Pique : Ces jeux préinstallés sous Windows 7 sont supprimés lors de l'installation de la mise à niveau vers Windows 10. Microsoft a publié sa version du Solitaire et du Démineur sous les noms de « Microsoft Solitaire Collection » et « Microsoft Minesweeper ».
    • Lecteurs de disquettes : Si vous possédez un lecteur de disquettes, vous devez télécharger le dernier pilote à partir de Windows Update ou du site Web du fabricant.
    • Si Windows Live Essentials est installé sur votre système, l'application OneDrive est supprimée et remplacée par la version boîte de réception de OneDrive.
    • Fichiers réservés OneDrive : Les fichiers réservés OneDrive ne sont pas pris en charge dans Windows 10. Windows 8.1 affichait des espaces réservés pour les fichiers disponibles dans OneDrive, mais pas en local sur l'appareil. Dans Windows 10, les utilisateurs peuvent choisir les dossiers à synchroniser depuis les paramètres OneDrive.
    • Ancrer : La fonctionnalité Ancrer est limitée à 2 applications en mode Tablette.
