    Sécurité de Windows

    Windows 10 change la donne en matière de sécurité en protégeant l'identité des utilisateurs, les périphériques et les informations grâce à une solution complète basée sur un système proposé exclusivement par Microsoft.

     | 

    Sécurité de Windows
    FonctionnalitéWindows 10 FamilleWindows 10 Professionnel

    En utilisant la reconnaissance faciale, les empreintes digitales ou un code PIN, Windows Hello est un moyen rapide, sécurisé et sans mot de passe pour déverrouiller vos appareils Windows compatibles.

    		

    Le chiffrement de l'appareil est disponible sur une large gamme de périphériques Windows et permet de protéger vos données en les chiffrant. Si vous activez le chiffrement de l'appareil, seules les personnes autorisées pourront accéder à votre périphérique et à vos données.

    		

    Votre appareil Windows est doté de fonctionnalités de sécurité avancées, y compris un pare-feu et des protections Internet pour vous protéger contre les virus, les programmes malveillants et les ransomware.

    		

    Le contrôle des applications et du navigateur dans Windows Security fournit les paramètres pour Windows Defender SmartScreen, qui permet de protéger votre appareil des applications, fichiers, sites web et téléchargements potentiellement dangereux.

    		

    Gérez le temps passé sur l'appareil, limitez l'accès au contenu pour adultes, contrôlez les achats en ligne lorsque vous connectez les comptes Microsoft de votre famille et vérifiez que tous vos appareils sont sécurisés et à jour.

    		

    Le démarrage sécurisé empêche les logiciels malveillants et les systèmes d'exploitation non autorisés de se charger pendant le processus de démarrage du système.

    		

    En cas de perte ou de vol de votre appareil, BitLocker et BitLocker To Go le verrouillent complètement afin que personne d'autre ne puisse accéder à votre système ou à vos données.

    		

    La protection des informations Windows (WIP) aide à se protéger contre les potentielles fuites de données sans interférer avec l'expérience de vos employés. Cette fonctionnalité permet également de protéger les applications et les données de l'entreprise contre les fuites de données accidentelles, que ce soit sur les périphériques appartenant à l'entreprise ou sur les périphériques personnels que les employés apportent au travail, et ce sans avoir à modifier votre environnement ou d'autres applications.

    		

    L'Antivirus Windows Defender utilise la puissance du cloud, l'optique large, le Machine Learning et l'analyse comportementale pour protéger vos périphériques des nouvelles menaces difficiles à bloquer.

    		
